Станимир Стайков

Я Frontend-dev! Я Разрабатываю приложение для детей с особенностями в развитии, в результате приложение стало востребовано, как для родителей «особых» детей, так и для терапевтов. Приложение состоит из мобильной части для пользователей и веб-сайта для администратора с возможностями контролирования пользователей. Возможности приложения: — запись видео сеансов, передача видеофайлов с приложения на веб-сайт; — запись данных вручную внутри приложения: время проведения терапии, симптомы и их тяжесть; — составление графиков анализа симптомов.
